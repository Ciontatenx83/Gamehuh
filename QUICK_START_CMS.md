## 🚀 Getting Started (2 minutes)

### Step 1: Access the Admin Panel
**Open:** `admin.html` in your browser

Or click **"Admin"** link in the navbar of your main site.

### Step 2: Add Your First Game
1. Go to **"Add New Game"** tab
2. Fill in the form:
   - Game Title
   - Developer Name
   - Choose Genre (Action, RPG, etc.)
   - Set Price
   - Write Description
   - Add Image URL
   - Add Download Link
3. Click **"Add Game"**

### Step 3: View Your Game
1. Go back to main site (index.html)
2. Scroll to "Browse All Games" section
3. 🎉 Your new game appears!

## 📌 What Can You Do?

### In the Store (index.html)
- Browse all games in a grid
- Click genre pills to filter (Action, RPG, Horror, etc.)
- Search games by title
- Click a game to see details
- Add games to cart

### In the Admin Panel (admin.html)

**Tab 1: Add New Game**
- Create new games with full details
- Auto-saves to browser storage

**Tab 2: Manage Games**
- See table of all games
- Edit any game by clicking pencil icon
- Delete games by clicking trash icon

**Tab 3: Game Statistics**
- View total games count
- See average price & rating
- Check breakdown by genre

**Tab 4: Settings**
- Export games to JSON file (backup)
- Import games from JSON file (restore)
- Reset to default games

## 💾 About Your Data

- **Stored:** In browser's localStorage
- **Persists:** Across page refreshes
- **Backup:** Export to JSON file
- **Restore:** Import JSON file
- **Safety:** Use export regularly!

### To Backup Your Games:
1. Admin Panel → Settings tab
2. Click **"Export Games (JSON)"**
3. Save the file somewhere safe

### To Restore Games:
1. Admin Panel → Settings tab
2. Click **"Import Games (JSON)"**
3. Select your backup file
4. Confirm

## ❓ Common Questions

**Q: How many games can I add?**
A: Thousands! Limited by browser storage (~5-10MB)

**Q: Can I edit games after adding?**
A: Yes! Admin → Manage Games → Click pencil icon

**Q: Will changes appear on the main site?**
A: Yes! Instantly, no reload needed

**Q: Can I delete a game?**
A: Yes! Admin → Manage Games → Click trash icon

**Q: How do I backup games?**
A: Admin → Settings → Click "Export Games (JSON)"

**Q: Can I import multiple games at once?**
A: Yes! Prepare a JSON file and import it
